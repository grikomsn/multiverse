## Get the game

- Download: https://gog.com/downloads/system_shock_classic/en1installer1
- Store page: https://gog.com/en/game/system_shock_enhanced_edition

## Extract game assets

- Download extractor: https://macsourceports.com/utility/extractor
- `mkdir -p "~/Library/Application Support/systemshock/res"`
- Copy extracted `DATA` and `SOUND` directories to `~/Library/Application Support/systemshock/res`

## Shockolate

- Download: https://macsourceports.com/game/systemshock (pick software mode)
- Extract and run Shockolate.app
